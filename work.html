LetsMeet

React, Node, oAuth, Serverless, PWA, Jest, Gherkin, Puppeteer


This serverless app utilizes React. Using oAuth, lambdas read a caledar with conference meeting times. It organizes the data and puts it in a searchable format along with charts showing the data in an easy-to-read format. Behind the scenes, there are a wide-range of testing strategies including TDD, BDD and End-to-End Acceptance Tests. While not as flashy as other apps, looking at the repository will show a lot of behind the scenes work which make it more responsive, robust and easy to improve on.
